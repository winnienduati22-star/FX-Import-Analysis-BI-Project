# FX Import Cost Analysis Dashboard

## Business Problem
This project analyzes exchange rate movements between EUR and four trading partners:

- USA (USD)
- UK (GBP)
- China (CNY)
- Kenya (KES)

The objective is to identify the most cost-effective country for imports and determine the optimal timing for international payments.

## Data Source
ExchangeRate API

## Technology Stack
- Python
- PostgreSQL
- Power BI
- SQLAlchemy
- Pandas

## Architecture

ExchangeRate API
↓
Python ETL
↓
PostgreSQL
↓
Power BI Dashboard
↓
Business Decision Support

## Dashboard Features
- FX Trend Analysis
- Country Comparison
- Import Timing Analysis
- Best Import Country KPI
- Historical Exchange Rate Monitoring
